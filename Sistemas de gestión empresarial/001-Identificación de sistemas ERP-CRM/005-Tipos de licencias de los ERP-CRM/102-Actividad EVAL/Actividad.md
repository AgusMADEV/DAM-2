# Actividad: Selección de Tipos de Licencias ERP-CRM

## Contexto

En un mundo empresarial donde la eficiencia y la toma de decisiones estratégicas son cruciales, los sistemas ERP-CRM (Enterprise Resource Planning - Planificación de Recursos Empresariales y Customer Relationship Management - Gestión Relacional de Clientes) juegan un papel fundamental. Estos sistemas optimizan la operativa diaria y facilitan el manejo de relaciones con clientes. Para seleccionar el sistema adecuado, es importante considerar su licenciatura, ya que esto puede influir significativamente en los costos totales de implementación y en las capacidades del software.

## Enunciado paso a paso

1. **Definición de la variable**: Define una variable llamada `tipo_licencia` que represente el tipo de licencia elegido para un sistema ERP-CRM.

2. **Selección del tipo de licencia**: Pide al usuario que indique su preferencia entre las tres opciones disponibles: "Privativas", "SaaS" o "Abiertas".

3. **Asignación del valor a la variable**: Dependiendo de la elección del usuario, asigna el valor correspondiente a la variable `tipo_licencia`.

4. **Mostrar la selección**: Imprime en pantalla el tipo de licencia seleccionado.

## Restricciones

- No utilizar funciones o métodos que no se hayan visto en clase.
- Solo usar los tipos de licencias mencionados: "Privativas", "SaaS" y "Abiertas".

## Información adicional sobre los tipos de licencias

### Licencias Privativas
Son licencias de software propietario donde el código fuente no está disponible para el usuario final. El software se vende con restricciones de uso y modificación.

### SaaS (Software as a Service)
En este modelo no se vende el software sino que se vende el acceso o el servicio. El software se ejecuta en la nube y los usuarios acceden a través de internet.

### Licencias Abiertas
Incluyen licencias como GPL, BSD, etc. El código fuente está disponible y puede ser modificado y redistribuido bajo ciertas condiciones.
